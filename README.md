# Haunted Room
A program egy interaktív 3D jelenetet valósít meg ahol a felhasználó egy sötét szobát járhat be.
A szobában különböző objektumok találhatók (asztal, láda, gyertya), illetve egy fáklya ami dinamikusan világítja be a szobát.

## Funkciók
- Kamera kezelés
- 3D modellek betöltése fájlból (.obj)
- Textúrázott objektumok
- Dinamikus fénykezelés
- Használati útmutató
- Köd
- Ütközésvizsgálat
- Interaktív objektum

## Technológia
- C programozási nyelv
- OpenGL
- GLFW
- GLAD
- stb_image

## Fordítás
A program a következő paranccsal fordítható: `make`
